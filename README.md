# Renegade Retro

Live page: [mysticalg.github.io/Renegade](https://mysticalg.github.io/Renegade/)

Arcade brawler prototype inspired by classic Renegade and Kunio-kun style street fights.

## Controls

- Move: Arrow keys or `A` / `D`
- Jump: `W` or Up Arrow
- Punch: `J`
- Kick: `K`
- Special: `L`

## Run locally

```bash
npm start
```

Or open `index.html` directly for a quick static preview.
